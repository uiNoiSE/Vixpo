ПРОСТРАНСТВО

Пользователь проекта (посетитель выставки) с помощью манипуляторов (клавиатура, мышь, геймпад, дисплей телефона) прогуливается по территории выставки и наблюдает классические рекламные стенды разных организаций.

  • Открытая площадка
    в виде парковой зоны
  • Индивидуальная реализация
    на основе реального
    мероприятия
  • Стандартный ЭкспоЦентр
    в виде большого помещения
    с высокими потолками
